Que es un entorno virtual?

Un entorno de desarrollo virtual python o simplemente entorno virtual python es un mecanismo que me permite gestionar programas y paquetes python sin tener permisos de administración.

Se utilizan mas para satisfacer proyectos DevOps .

		 Es un marco de trabajo y una filosofía en constante evolución que promueve un mejor desarrollo de aplicaciones en menos tiempo y la rápida publicación de nuevas o revisadas funciones de software o productos para los clientes.

Como crear y activar un entorno virtual.

virtualenv 'Nombre'
'Nombre'\Scripts\activate


Y listo calisto!! <3

Nota adicional.

Librerias instaladas para este ejemplo!!
